### About the Project
**Duck Game Studio - Solitaire** is a cross-platform card game project. It marks the first contribution to the open-source game ecosystem by a developer with **7 years of experience in embedded engineering**.

Unlike typical UI-based card games, this project leverages the precision and efficiency often required in embedded systems to deliver a smooth gaming experience across all devices.

### Key Features
*   **ğŸš€ Hardware-Optimized Rendering:** Designed with an embedded engineer's mindset, focusing on lightweight and fast graphics rendering.
*   **ğŸ¨ SkiaSharp Integration:** Utilizes SkiaSharp for high-performance 2D graphics instead of standard UI controls.
*   **ğŸ“± True Cross-Platform:** Runs seamlessly on **Windows, macOS, Android, and iOS** using a single codebase.

### Tech Stack
*   **Framework:** .NET MAUI
*   **Graphics:** SkiaSharp
*   **Architecture:** MVVM (CommunityToolkit.Mvvm)

### Installation & Run
You can build and run the project using the .NET CLI:

```bash
dotnet build
```
